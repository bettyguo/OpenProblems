### D-4. `renderReviewNotesMarkdown` shape

**Decision**: new sibling helper in `lib/markdown/index.ts` per
ADR-0018 D-G inheritance contract.

```ts
export function renderReviewNotesMarkdown(text: string | null): string | null {
  if (text === null) return null;
  const trimmed = text.trim();
  if (trimmed.length === 0) return null;
  const file = reviewNotesProcessor.processSync(trimmed);
  const html = String(file).trim();
  return html.length === 0 ? null : html;
}
```

Singleton `reviewNotesProcessor` instance with identical pipeline
to `bioProcessor` (renamed from Phase-17's `processor` for clarity).
Both processors share the same 7-stage pipeline + custom plugins
(`rehypeDemoteHeadings` + `rehypeStripUnsafeHrefs`).

**Refactoring**: Phase 17's anonymous `processor` constant renamed
to `bioProcessor` to disambiguate the pair. No behavior change; one
identifier rename.
