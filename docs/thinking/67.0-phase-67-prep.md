### Phase-67-blocking decisions deferred to per-unit implementation

- **D-8** (Unit 67.1): className value — hardcoded literal
  `["wikilink", "wikilink-unresolved"]` (no curator
  parameterization at Phase 67). Future Phase 68+ curator-
  configurable variant deferred via plugin-option axis 4th
  realization. Lean = **hardcoded array**.
- **D-9** (Unit 67.1): className array order — `["wikilink",
  "wikilink-unresolved"]` (primary class first; modifier
  second). Mirrors BEM-style class ordering convention. Lean
  = **primary-first**.
- **D-10** (Unit 67.1): `isValidTarget` predicate when target
  resolves true — emit single-class `["wikilink"]` (Phase-65
  baseline) NOT `["wikilink", "wikilink-resolved"]` (would be
  redundant; CSS can use `.wikilink:not(.wikilink-unresolved)`
  selector pattern). Lean = **emit single-class when
  resolved**.
- **D-11** (Unit 67.1): backward-compat default behavior when
  `isValidTarget` is undefined — emit single-class
  `["wikilink"]` (Phase-65 behavior preserved verbatim).
  Mirrors Phase-62 `buildHref?` default. Lean = **preserve
  Phase-65 default**.
- **D-12** (Unit 67.2): Velite collections to traverse for
  valid-targets — `problems` (problemSlugs) + `papers`
  (paperIds) + `authors` (authorSlugs) + `institutions`
  (institutionSlugs). Mirrors the Phase-66 audit-script set
  exactly — drift-free. Lean = **same 4 collections**.
- **D-13** (Unit 67.2): module-init-time-load vs per-request-
  load for valid-targets — module-init-time (Velite content
  is in-bundle; load is O(N) per process; amortized across
  requests). Lean = **module-init-time**.
- **D-14** (Unit 67.2): new arm name — `wikilinks-validated`
  (semantic; communicates "renders fallback className when
  build-time validation didn't catch the case"). Alternative
  names considered: `wikilinks-with-fallback` (verbose);
  `wikilinks-defense-in-depth` (overlong); `wikilinks-strict`
  (misleading — doesn't reject, just visually marks). Lean =
  **`wikilinks-validated`**.
- **D-15** (Unit 67.1): APPEND-D-AY naming — Excel
  convention continues (D-AX → D-AY). Lean = **continue
  Excel**.
- **D-16** (Unit 67.1): tuple-form vs bare-form plugin emit
  in `WikilinkExtensionRegistry.getExtensions` when ONLY
  `isValidTarget` is set (without `buildHref`) — emit tuple-
  form `[rehypeResolveWikilinks, { isValidTarget }]` (mirrors
  Phase-63 cross-entity tuple-form when `buildHref` is set).
  Lean = **tuple-form for any non-default option**.
