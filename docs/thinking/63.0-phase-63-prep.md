### D-3. Cross-entity wikilinks shape anticipated (Unit 63.1 + 63.2)

D-A. **`WIKILINK_PATTERN` regex extension** (Unit 63.1):

```ts
// Before (Phase 46 ship through Phase-62 close):
const WIKILINK_PATTERN = /\[\[([a-z0-9-]+)(?:\|([^\]\n]+))?\]\]/g;

// After (Phase 63 ship):
const WIKILINK_PATTERN = /\[\[(?:([a-z0-9-]+):)?([a-z0-9-]+)(?:\|([^\]\n]+))?\]\]/g;
```

Capture groups:
- Group 1: optional `entity-type` (e.g., `paper`, `author`,
  `institution`); `undefined` when wikilink uses bare
  `[[slug]]` syntax.
- Group 2: `slug` (e.g., `arxiv-2401-12345`, `jane-doe`,
  `mit`); always defined.
- Group 3: optional `display` alias (Phase-46 unchanged).

**Backwards-compat preserved**: every Phase-38 / Phase-46
wikilink shape continues to match the extended regex with
`entityType = undefined`:
- `[[plain-slug]]` → group 1 undefined, group 2 = "plain-slug".
- `[[plain-slug|display]]` → group 1 undefined, group 2 =
  "plain-slug", group 3 = "display".

New Phase-63 shapes:
- `[[paper:arxiv-2401-12345]]` → group 1 = "paper", group 2 =
  "arxiv-2401-12345".
- `[[author:jane-doe|Jane Doe]]` → group 1 = "author", group 2
  = "jane-doe", group 3 = "Jane Doe".
- `[[institution:mit]]` → group 1 = "institution", group 2 =
  "mit".

`DEFAULT_BUILD_HREF` remains byte-identical to Phase-38

D-D. **`CROSS_ENTITY_BUILD_HREF` NEW** (Unit 63.1):

```ts
export const CROSS_ENTITY_BUILD_HREF = (slug: string, entityType?: string): string => {
  // Phase 63 cross-entity router. Routes by entity-type prefix
  // captured from regex `(?:([a-z0-9-]+):)?` group. `entityType`
  // undefined for bare `[[slug]]` syntax (preserves Phase-38
  // baseline routing).
  if (entityType === "paper") return `/papers/${slug}`;
  if (entityType === "author") return `/authors/${slug}`;
  if (entityType === "institution") return `/institutions/${slug}`;
  // Default arm: undefined OR unknown entity-type → /problems/${slug}.
  // Unknown entity-types are gracefully degraded (the regex captures
  // any `[a-z0-9-]+:` prefix; the builder routes only on the three
  // known types). Curators who typo an entity-type get the same
  // fallback behavior as bare wikilinks.
  return `/problems/${slug}`;
};
```

**XSS-safety contract preserved**: `slug` is constrained to
`[a-z0-9-]+` per APPEND-D-I (the regex IS the validation);
`entityType` is similarly constrained to `[a-z0-9-]+` by the
regex. Builders that interpolate these values into URL paths
preserve the XSS-safety contract automatically.

D-E. **`WikilinkExtensionRegistry` constructor evolution**
(Unit 63.2):

```ts
// Before (Phase 38 → Phase-62 close):
export class WikilinkExtensionRegistry implements MarkdownExtensionRegistry {
  private readonly enabledSurfaces: ReadonlySet<MarkdownSurface>;

  constructor(enabledSurfaces: ReadonlySet<MarkdownSurface>) {
    this.enabledSurfaces = enabledSurfaces;
  }

  getExtensions(surface: MarkdownSurface): MarkdownExtensionSet {
    if (this.enabledSurfaces.has(surface)) {
      return { rehypePlugins: [rehypeResolveWikilinks] };
    }
    return {};
  }
}

// After (Phase 63 ship):
export class WikilinkExtensionRegistry implements MarkdownExtensionRegistry {
  private readonly enabledSurfaces: ReadonlySet<MarkdownSurface>;
  private readonly buildHref: ResolveWikilinksOptions["buildHref"];

  constructor(
    enabledSurfaces: ReadonlySet<MarkdownSurface>,
    options: { buildHref?: ResolveWikilinksOptions["buildHref"] } = {},
  ) {
    this.enabledSurfaces = enabledSurfaces;
    this.buildHref = options.buildHref;
  }

  getExtensions(surface: MarkdownSurface): MarkdownExtensionSet {
    if (!this.enabledSurfaces.has(surface)) return {};
    if (this.buildHref === undefined) {
      // Phase 38-62 bare-form emit preserved when no buildHref is set.
      return { rehypePlugins: [rehypeResolveWikilinks] };
    }
    // Phase 63+ tuple-form emit when buildHref is set.
    return { rehypePlugins: [[rehypeResolveWikilinks, { buildHref: this.buildHref }]] };
  }
}
```

**Backwards-compat preserved**: existing factory call
`new WikilinkExtensionRegistry(PHASE_38_DEFAULT_ENABLED_SURFACES)`
continues to work (defaults to `options = {}` → `buildHref =
undefined` → bare-form emit). The `MARKDOWN_EXTENSIONS=wikilinks`
arm behavior is byte-identical to Phase 62.
