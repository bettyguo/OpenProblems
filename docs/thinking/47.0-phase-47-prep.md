  collision. AND wikilinks runs in `rehypePlugins` stage AFTER

### D-3. arxiv alias regex shape anticipated (Unit 47.1)

D-A. **`ARXIV_PATTERN` regex evolution** (in-place edit of
`lib/markdown/extensions/arxiv.ts`):

```ts
// Before (Phase 41 ship through Phase-46 close):
const ARXIV_PATTERN = /\barxiv:(\d{4}\.\d{4,5})(v\d+)?\b/gi;

// After (Phase 47 ship):
const ARXIV_PATTERN =
  /\[\[arxiv:(\d{4}\.\d{4,5})(v\d+)?(?:\|([^\]\n]+))?\]\]|\barxiv:(\d{4}\.\d{4,5})(v\d+)?\b/gi;
```

**First dual-form regex** in the framework: alternation between
the bracketed form (priority) and the bare form (fallback).
Engine tries the bracketed alternative first; if it fails at a
given position, tries the bare alternative.

- **Bracketed form** `\[\[arxiv:(...)(...)?(?:\|([^\]\n]+))?\]\]`:
  - Group 1 = arxiv ID (4-digit-prefix + 4-5-digit-suffix; same
    as Phase-41 baseline).
  - Group 2 = optional version (`v\d+`; same as Phase-41
    baseline).
  - Group 3 = optional display text (`[^\]\n]+`; same shape as
    Phase-46 wikilinks alias).
- **Bare form** `\barxiv:(...)(...)?\b`:
  - Group 4 = arxiv ID (Phase-41 baseline preserved).
  - Group 5 = optional version (Phase-41 baseline preserved).
  - No alias support in bare form (alias requires bracket-
    wrapping per APPEND-D-Y item 5 deferral text).

D-B. **`remarkLinkArxivIds` plugin body update**:

```ts
const isBracketed = match[0].startsWith("[[");
const id = isBracketed ? match[1] : match[4];
const version = (isBracketed ? match[2] : match[5]) ?? "";
const alias = match[3]; // only set for bracketed form
if (id === undefined) continue;

const url = `https://arxiv.org/abs/${id}${version}`;
let display: string;
if (alias) {
  display = alias;
} else if (isBracketed) {
  // Bracketed form without alias: display the verbatim arxiv
  // reference minus brackets (e.g., `[[arxiv:1909.03004]]` →
  // `<a>arxiv:1909.03004</a>`).
  display = match[0].slice(2, -2);
} else {
  // Bare form (Phase-41 baseline): display source casing
  // verbatim (e.g., `ArXiv:2024.12345`).
  display = match[0];
}

newNodes.push({
  type: "link",
  url,
  children: [{ type: "text", value: display }],
});
```
