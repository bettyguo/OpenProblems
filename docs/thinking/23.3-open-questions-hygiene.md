### Implementation-note convention update

Phase 22 (Unit 22.3) annotated ADR-0014 D-D with a "REALIZED
Phase 22 Unit 22.1" note. Phase 23 (this unit) appends an
"ENHANCED Phase 23 Unit 23.1" note to the same paragraph rather
than amending the Phase-22 note in-place. This preserves the
phase-by-phase audit trail (per ADR-0005 immutability ethos
applied loosely to ADR amendments) and keeps each phase's
contribution distinguishable.

**Convention now**: each phase that touches a previously-pinned
ADR D-clause APPENDS a phase-tagged note rather than EDITING
prior phase notes. Mirrors ADR-0005 rating-action immutability
in spirit — past phase contributions are immutable signatures
of state-at-that-time.
