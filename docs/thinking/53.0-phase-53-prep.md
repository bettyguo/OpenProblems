### D-3. Arxiv legacy regex shape anticipated (Unit 53.1)

D-A. **`ARXIV_PATTERN` regex evolution**:

```ts
// Before (Phase 47 ship through Phase-52 close):
const ARXIV_PATTERN =
  /\[\[arxiv:(\d{4}\.\d{4,5})(v\d+)?(?:\|([^\]\n]+))?\]\]|\barxiv:(\d{4}\.\d{4,5})(v\d+)?\b/gi;

// After (Phase 53 ship):
const ARXIV_PATTERN =
  /\[\[arxiv:(\d{4}\.\d{4,5}|[a-z]+(?:-[a-z]+)*(?:\.[A-Z-]+)?\/\d{7})(v\d+)?(?:\|([^\]\n]+))?\]\]|\barxiv:(\d{4}\.\d{4,5}|[a-z]+(?:-[a-z]+)*(?:\.[A-Z-]+)?\/\d{7})(v\d+)?\b/gi;
```

**Inner ID-class disjunction** added in BOTH the bracketed
alternative AND the bare alternative. The ID-class becomes:

- **Modern** `\d{4}\.\d{4,5}` (Phase 41 baseline; e.g.,
  `1909.03004`, `2024.01234`).
- **Legacy** `[a-z]+(?:-[a-z]+)*(?:\.[A-Z-]+)?/\d{7}` (Phase
  53 NEW):
  - **Category** `[a-z]+(?:-[a-z]+)*` — matches single-word
    categories (`math`, `physics`) AND hyphenated multi-word
    categories (`hep-th`, `cond-mat`, `nucl-ex`, `astro-ph`).
  - **Subcategory** `(?:\.[A-Z-]+)?` — optional uppercase
    subcategory (`cs.AI`, `cs.GT`, `cs.IT`); supports
    hyphenated subcategories.
  - **Separator** `/` — literal slash.
  - **ID** `\d{7}` — exactly 7 digits (year-month + paper
    number; e.g., `0211159` = November 2002, paper 159).

**Examples** post-Phase-53:

| Form | Match? | Notes |
|---|---|---|
| `arxiv:1909.03004` | YES | Modern; Phase-41 baseline preserved |
| `arxiv:2024.01234` | YES | Modern with 5-digit suffix; Phase-41 baseline |
| `arxiv:math/0211159` | YES | NEW Phase-53; pre-2007 simple category |
| `arxiv:cs.AI/0501001` | YES | NEW Phase-53; subcategory format |
| `arxiv:hep-th/9711200` | YES | NEW Phase-53; hyphenated category |
| `arxiv:cs.GT/0309136` | YES | NEW Phase-53; uppercase subcategory |
| `arxiv:cond-mat/0301001v3` | YES | NEW Phase-53; with version suffix |
| `[[arxiv:math/0211159\|Witten 2002]]` | YES | NEW Phase-53; bracketed alias for legacy ID |
| `arxiv:foo/0211159` | YES | NEW Phase-53 admits ANY lowercase category prefix; over-permissive but matches arXiv's permissive historical category list |
| `arxiv:Math/0211159` | NO | Uppercase category rejected (categories are lowercase) |
| `arxiv:math/021115` | NO | 6 digits rejected (legacy IDs are exactly 7) |
| `arxiv:math/02111590` | NO partial | Backs off to 7-digit match; trailing 0 outside |

**Version suffix `(v\d+)?`** preserved verbatim; applies to
both modern and legacy formats (arXiv supports version suffixes
across formats).

**Inner-class disjunction discipline**: the alternation
`\d{4}\.\d{4,5}|[a-z]+(?:-[a-z]+)*(?:\.[A-Z-]+)?\/\d{7}` is
unambiguous because the two alternatives have disjoint first
characters (`\d` vs `[a-z]`) — no backtracking ambiguity.
