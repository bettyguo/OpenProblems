   deferral. Phase 17+ if bandwidth concerns surface (large
   originals served on every render); `sharp` on Vercel
   serverless. ~3-4 units.

   deferral. Phase 17+ if curator demands editorial

   `userImageHistory` table + migration. ~3 units.

   (square-ratio enforcement). Phase 17+ if UX feedback

   moderation prudence).
