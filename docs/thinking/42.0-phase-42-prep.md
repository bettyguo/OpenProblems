### D-4. APPEND letter sequence convention

Phase 41 added APPEND-D-U through APPEND-D-Y (last single-
letter remaining: Z). Phase 42 has two options for the next
letters:

- **Option α**: APPEND-D-Z (single-letter continuation; only
  one letter remains until the alphabet cycle wraps).
- **Option β**: APPEND-D-AA + APPEND-D-AB + ... (two-letter
  block continuation; common convention when single-letter
  space is exhausted).

Phase 42 anticipates only 1-3 APPEND letters needed (smaller
scope than Phase 41's 5 APPENDs). Lean: **Option α** (use
the remaining single letter D-Z for the primary expansion-
shape APPEND); if a Phase-43+ APPEND is needed, the
single-letter alphabet wraps to **Option β** at that point.
The audit-trail-friendly precedent (single letters used in
order) carries.
