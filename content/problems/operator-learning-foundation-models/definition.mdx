---
title: Formal definition
summary: Solution operators as the learning target; in-distribution accuracy versus out-of-distribution generalisation as the two distinct quantities a benchmark must measure.
---

Let a parametric PDE family be specified by a parameter set $\Theta$, an initial-condition space $\mathcal{U}_0$, and a domain $\Omega$. The *solution operator* $\mathcal{G}_\theta : \mathcal{U}_0 \to \mathcal{U}_T$ takes an initial state $u_0$ to the state at time $T$ under parameters $\theta \in \Theta$.

A **neural operator surrogate** is a learned $\hat{\mathcal{G}}_\phi$ trained on samples $(u_0^{(i)}, u_T^{(i)})$ generated by a classical solver, parameterised by neural-network weights $\phi$. The natural error on a test distribution $\mathcal{D}$ is

$$
\mathcal{E}_{\mathrm{in}}(\hat{\mathcal{G}}_\phi, \mathcal{D}) \;=\; \mathbb{E}_{(u_0, u_T) \sim \mathcal{D}} \, \big\| \hat{\mathcal{G}}_\phi(u_0) - u_T \big\|.
$$

**Generalisation gap.** Define a shifted distribution $\mathcal{D}'$ — different parameter regime, geometry, or initial-condition class. The *generalisation gap* is $\mathcal{E}_{\mathrm{out}}(\hat{\mathcal{G}}_\phi, \mathcal{D}') - \mathcal{E}_{\mathrm{in}}(\hat{\mathcal{G}}_\phi, \mathcal{D})$. A solved version of this problem makes that gap small — close to zero relative to the in-distribution error — across the kinds of distribution shift that matter for the application (extreme events, unseen geometries, parameter extrapolation).
